### 🧠 What I'm building

**[Kit for AI](https://kitforai.com)** — a second brain for AI agents. Turn any document, page, or conversation into clean, searchable, *cited* knowledge, and give every agent a memory that persists across sessions — over **MCP**, **REST**, and a typed **SDK**, with one API key.

- 📄 Document & URL → Markdown / JSON conversion (with OCR)
- 📚 Knowledge bases with hybrid semantic search, built for RAG
- 🧩 `remember` / `recall` agent memory over MCP
- 💬 Grounded chat that cites its sources

Explore the open-source developer hub → **[github.com/kitforai](https://github.com/kitforai)**
